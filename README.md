# SKATE Dash

SKATE Dash visualizes the json data from [SKATE3](https://github.com/born-physical-studied-digitally/SKATE3_2) along with the seismogram image. It is built on plotly and dash to enable local as well as web deployment.

Plotting attrbutes are customizable in the config.yml file.

See the AI generated claude.md file for more details.

# Installation

See INSTALL.md for the list of dependencies.

# Contributing

All contributions are welcome. 

To make a change to you should:

* Create a fork (through GitHub) of the code base.
* Create a separate branch (sometimes called a feature branch) on which you do your modifications.
* You can propose that your branch be merged by opening a pull request. This will give others a chance to review your code.

# Use of AI
SKATE was prototyped and structured initially wihout the use of AI. Subsequently, AI aided coding has been used to add new features.  This includes Chat GPT 4.1 or later and Claude Sonnet 4.6 or later.

All use of AI must be disclosed.

All AI generated code must be human reviewed before submitting a pull request.

# Development Plan
SKATE_Dash is currently developed as a visualization program with limited interactive features. At some point it may be extended for interactive editing of json features.
